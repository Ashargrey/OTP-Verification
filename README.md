## 🚀 Features

- 📧 Send OTP to user's email using EmailJS
- 🔐 Verify OTP instantly on frontend
- 🎨 Modern UI (Glassmorphism design)
- 🔔 Toast notifications (no alerts)
- ⏳ Loader animation while sending OTP
- ⚡ Fast and responsive design
- 🧠 Random 6-digit OTP generation

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Modern UI + Animations)
- JavaScript (Core Logic)
- EmailJS (Email sending without backend)

---

## ⚙️ How It Works

1. User enters email
2. System generates a random OTP
3. OTP is sent via EmailJS
4. User enters OTP
5. System verifies OTP
